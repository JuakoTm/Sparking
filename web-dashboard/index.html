Preguntas Frecuentes

¿Cómo funciona el sistema?

Cada puesto cuenta con un sensor inteligente que detecta la presencia del vehículo y comunica el estado a nuestra plataforma cloud. El dashboard web muestra todo en tiempo real y los conductores ven señales visuales (luces) para encontrar espacios disponibles.

¿Necesito personal técnico para administrarlo?

No. El dashboard es 100% web, accesible desde cualquier navegador. Los administradores pueden crear zonas, agregar puestos, ver estadísticas y gestionar reservas sin conocimientos técnicos previos.

¿Qué pasa si se corta internet?

El hardware cuenta con lógica local. Las señales visuales (luces) siguen funcionando de forma autónoma. Una vez restaurada la conexión, los datos se sincronizan automáticamente con la nube.

¿Es escalable?

Totalmente. Puedes comenzar pilotando una zona pequeña (ej. 20 puestos) y expandir progresivamente sin cambiar la infraestructura. El dashboard soporta múltiples zonas y miles de puestos sin pérdida de rendimiento.
